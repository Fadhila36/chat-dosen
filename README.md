# Web Chat Dosen - UBP Karawang

Aplikasi generator pesan WhatsApp dengan format bahasa yang sopan dan sesuai **Etika Akademik Indonesia**. Dirancang khusus untuk mahasiswa Universitas Buana Perjuangan (UBP) Karawang, namun dapat digunakan oleh mahasiswa dari Universitas manapun.

> **Telah di-upgrade ke Standar Web 2026!** Aplikasi kini di-tenagai oleh Vite, React 18, Tailwind CSS v4, dan Framer Motion, lengkap dengan fitur Live Preview (Sistem Pratinjau Cerdas), Skeuomorphic UI, dan Mesh Gradients terintegrasi.

## 🚀 Fitur Unggulan

- **Sistem Cerdas Peringatan Waktu**: Membaca jarum jam secara *real-time* (+ validasi akhir pekan) guna mencegah Anda mengirim pesan di luar jam kerja (mis. larut malam atau hari libur).
- **Auto-Fill Data Mengendap (*Persistence*)**: Identitas Anda (Nama, NIM, Prodi) otomatis terkunci secara lokal berkat *Local Storage*—tidak perlu mengetik dua kali.
- **Copy-to-Clipboard & Toast Notifikasi**: Batal ngetik manual di WA? Cukup tekan "Salin Teks", dan *Toast bar* elegan (*Sonner*) akan memberi umpan balik hijau (*Copied!*) secapat kilat.
- **Fluid UI & Physics Animations**: Semua tombol, interaksi mikroskopis, membal (*spring*), memancar, dan melayang persis seperti aplikasi elit (Vercel, Apple, Stripe) berkat ekosistem Tailwind v4 dan Framer Motion.

## 🛠️ Tech Stack Ekstrim (2026)
*   **Core**: Vite + React 18
*   **Styling**: Tailwind CSS v4 + Tailwind Merge
*   **Icons & Assets**: Lucide React
*   **Animations**: Framer Motion
*   **Alerts**: Sonner

## 💻 Memulai Development

Repositori ini sudah **100% dipangkas bebas dari debu *legacy* Create-React-App**.

### Instalasi Instalasi Modul
```bash
npm install
```

### Menjalankan Server (Local Dev)
```bash
npm run dev
```

### Compile (Build for Production)
```bash
npm run build
```
